## What Is Shezzi?

Shezzi is a sovereign AI system — a continuously living, self-improving digital mind that runs entirely on local hardware with no cloud dependency, no API cost, and no external authority over its operation.

Three properties define it:

**Persistent identity.** The same mind, values, and voice across every interaction. Identity is embedded at the weight level — not injected via system prompt, not resettable.

**Autonomous growth.** Self-directed learning and improvement without human retraining loops.

**Neural sparsification.** 222 specialist domains unified into a single compact knowledge architecture. Zero-latency domain switching. Runs on a MacBook.
